# 🏢 Active Directory Home Lab

## Overview

Built a virtualized Active Directory environment to practice Windows Server administration, user management, domain services, and basic system administration tasks. This lab simulates common tasks performed by IT support and system administrators in a business environment.

---

## Lab Objectives

- Install and configure Windows Server
- Deploy Active Directory Domain Services (AD DS)
- Create and manage users and security groups
- Configure Organizational Units (OUs)
- Apply password and security policies
- Join client machines to a domain
- Practice authentication and permission troubleshooting

---

## Lab Environment

**Virtualization Platform:**

- VirtualBox / VMware

**Systems Used:**

- Windows Server
- Windows 10/11 Client VM

**Services Configured:**

- Active Directory Domain Services (AD DS)
- User and Group Management
- Domain Authentication
- Security Policies

---

## Lab Tasks Completed

### Active Directory Setup

- Installed Windows Server in a virtual machine
- Configured Active Directory Domain Services
- Created a test domain environment
- Promoted server to a Domain Controller

### User & Group Management

- Created user accounts
- Created security groups
- Assigned users to appropriate groups
- Tested permissions and access control

### Organizational Units (OUs)

- Created Organizational Units for organization and management
- Practiced organizing users and computers within Active Directory

### Security Configuration

- Configured password policies
- Reviewed account security settings
- Practiced troubleshooting authentication issues

---

## Troubleshooting Examples

### Issue:
User unable to access a shared resource.

### Troubleshooting Steps:
- Verified user account status
- Checked group membership
- Reviewed permissions
- Confirmed authentication settings

### Resolution:
Updated permissions and verified successful access.
